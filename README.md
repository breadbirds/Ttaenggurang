## ✨ 기술적 특징

### 1. K-Means Clustering 모델 활용

-   비 지도 학습 -> 지도학습 순서로 진행
-   비 지도 학습을 통해 비슷한 유형끼리 군집화하여 소비 유형 분류 (소비형, 저축형, 투자형)
-   지도학습을 추가로 진행하여 예측 데이터를 제공하고, 분류가 얼마나 잘 이루어지는지 테스트 98.89% 정확도 결과.
